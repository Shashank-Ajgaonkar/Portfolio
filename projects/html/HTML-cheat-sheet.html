HTML Cheat Sheet

Introduction to HTML
    What is HTML?
    • HTML stands for Hyper Text Markup Language.

    • HTML is the standard markup language for creating Web pages.

    • HTML describes the structure of a Web page.

    • HTML consists of a series of elements.E.g: head,body,p,img etc.

    • HTML elements tell the browser how to display the content.

    • The HTML elements are NOT case sensitive.

    • Your web browser (e.g. Chrome,Safari, etc.) can read HTML code and translate it into web pages.

    What is an HTML element?

    HTML syntax
    • The standards and best practices are needed to make:
      1. web pages compatible with different browser and devices.
      2. teamwork and collaboration more effective.

    • Indentation is considered a good practice to improve code readibility.
      Indentation is the spaces at the beginning of lines.

      • !doctype declaration
        1. All HTML documents must start with a !DOCTYPE declaration.
        2. The declaration is not an HTML tag.
        3. It is an information to the browser about what document type to expect.
        4. The !DOCTYPE declaration is NOT case sensitive.

      • html tag
        1. The html tag represents the root of an HTML document.
        2. The html tag is the container for all other HTML elements (except for the !DOCTYPE tag).
      • head tag
        1. The head tag needs to be places before the body tag.
        2. The head tag is used to include technical information about the web page.
        3. The information in the head tag is not displayed on the web page. Only the title is shown in search engine result (e.g. Google) and the browser tab.

      • body tag
        1. The body tag defines the document's body.
        2. The body element contains all the contents of an HTML document, such as headings, paragraphs, images, hyperlinks, tables, lists etc.
        3. Note: There can only be one body element in an HTML document.

HTML Basic
    Text formatting tags

    1. The p tag defines a paragraph.

    2. Browsers automatically add a single blank line before and after each p element.

    3. The br tag inserts a single line break.

    4. The br tag is an empty tag which means that it has no end tag.

    5. Use the br tag to enter line breaks, not to add space between paragraphs.

    6. The b tag specifies bold text without any extra importance.

    7. The i tag defines a part of text in an alternate voice or mood. The content inside is typically displayed in italic.

    8. The u tag represents some text that is unarticulated and styled differently from normal text. The content inside is typically displayed with an underline.

    9. The sub tag defines subscript text. Subscript text appears half a character below the normal line,
      and is sometimes rendered in a smaller font. Subscript text can be used for chemical formulas, like H2O.

    10. The sup tag defines superscript text. Superscript text appears half a character above the normal line,
      and is sometimes rendered in a smaller font. Superscript text can be used for footnotes, like 22= 4.

    Headings tags

    1. The text on web pages can be organized in different sections with headings.

    2. Headings in HTML come in different levels. h1 defines the most important heading.

    3. You can use up to 6 levels of headings in HTML. The tags for these heading elements are h1 to h6.

    4. Headings are another element that requires both opening and closing container tags.

    5. You can add different headings to your page to organize the content.
      h1 is usually styled with the largest font and h6 with the smallest in order.

    6. You should only have one level 1 heading per page. Heading levels need to be used in correct order
      i.e. after your h1 you can use as many h2s as you need, after each h2 you can include h3 and so on.

    Image mapping and marquee

    1. The map tag is used to define an image map. An image map is an image with clickable areas.

    2. The required name attribute of the map element is associated with the img tag's usemap attribute and creates a relationship between the image and the map.

    3. The map element contains a number of area elements, that defines the clickable areas in the image map.

    4. Some of the important attribute in area tag are:
      • shape - Specifies the shape of the clickable area. The shape can be default,rect,circle,poly.
      • coords - Specifies the coordinates of the area.
      • href - Specifies the hyperlink target for the area.
      • target - Specifies where to open the target URL.

Advance HTML
    Table
    1. HTML tables allow web developers to arrange data into rows and columns.

    2. A table in HTML consists of table cells inside rows and columns.

    3. Each table cell is defined by a td container tag. td stands for table data.

    4. Each table row starts with a tr container tag. tr stands for table row.

    5. If you want your cells to be table header cells. In those cases use the th tag instead of the td tag. th stands for table header.

    6. By default, the text in th elements are bold and centered, but you can change that with CSS.

      7. Two important attributes in table are:

      • Cell Padding - Cell padding is the space between the cell edges and the cell content. By default the padding is set to 0.

      • Cell Spacing - Cell spacing is the space between each cell. By default the space is set to 2 pixels.

    Form

    1. The form tag is used to create an HTML form for user input.

      2. The form element can contain one or more of the following form elements:

        1. input tag:

        • The input tag specifies an input field where the user can enter data.

        • The input element is the most important form element.

        • The input element can be displayed in several ways, depending on the type attribute.
          Some of the type being button,checkbox,radio,date,range,email,password,text(default value) etc.


        2. textarea tag:

        • The textarea tag defines a multi-line text input control.

        • The textarea element is often used in a form, to collect user inputs like comments or reviews.

        • A text area can hold an unlimited number of characters, and the text renders in a fixed-width font (usually Courier).

        • The size of a text area is specified by the cols and rows attributes (or with CSS).

        • The name attribute is needed to reference the form data after the form is submitted
          (if you omit the name attribute, no data from the text area will be submitted).

        • The id attribute is needed to associate the text area with a label.


        3. select tag:

        • The select element is used to create a drop-down list.

        • The select element is most often used in a form, to collect user input.

        • The name attribute is needed to reference the form data after the form is submitted
          (if you omit the name attribute, no data from the drop-down list will be submitted).

        • The id attribute is needed to associate the drop-down list with a label.

        • The option tags inside the select element define the available options in the drop-down list.


        4. option tag:

        • The option tag defines an option in a select list.

        • The option tag can be used without any attributes, but you usually need the value attribute,
          which indicates what is sent to the server on form submission.

        • For reference:




        5. fieldset tag:

        • The fieldset tag is used to group related elements in a form.

        • The fieldset tag draws a box around the related elements.

        • The legend tag defines a caption for the fieldset element.


        6. label tag:

        • The label tag defines a label for several elements.

        • A label can also be bound to an element by placing the element inside the label element.

        • The for attribute of label must be equal to the id attribute of the related element to bind them together.

          • Proper use of labels with the elements above will benefit:

          1. Screen reader users (will read out loud the label, when the user is focused on the element).

          2. Users who have difficulty clicking on very small regions (such as checkboxes) -
            because when a user clicks the text within the label element, it toggles the input (this increases the hit area).

    Audio / Video

      1. Audio tag:

      • The audio tag is used to embed sound content in a document, such as music or other audio streams.

      • The audio tag contains one or more source tags with different audio sources. The browser will choose the first source it supports.

      • The text between the opening and closing audio tags will only be displayed in browsers that do not support the audio element.

      • There are three supported audio formats in HTML: MP3, WAV, and OGG.


      2. Video tag:

      • The video tag is used to embed video content in a document, such as a movie clip or other video streams.

      • Similar to audio tag, the video tag can contain multiple source tags and the browser will choose the first source it supports,
        and will display any content it has if the browser does not support the video element.

      • There are three supported video formats in HTML: MP4, WebM, and OGG.

    Inline and Block elements

    1. Every HTML element has a default display value, depending on what type of element it is.

    2. The two most common display values are block and inline.

    3. A block-level element always starts on a new line, and the browsers automatically add some space (a margin) before and after the element.

    4. A block-level element always takes up the full width available (stretches out to the left and right as far as it can).

    5. The commonly used block elements are: p, div, form, ul, ol, video, table etc.

    6. An inline element does not start on a new line.

    7. An inline element only takes up as much width as necessary.

    8. An inline element cannot contain a block-level element.

    9. The commonly used inline elements are: a, br, i, img, sub, sup, input, map, label, span etc.

    Div and span elements

    1. The div element is often used as a container for other HTML elements.

    2. The div element has no required attributes, but style, class and id are common.

    3. When used together with CSS, the div element can be used to style blocks of content.

    4. The span element is an inline container used to mark up a part of a text, or a part of a document.

    5. The span element just like div element has no required attributes, but style, class and id are common.

    6. When used together with CSS, the span element can be used to style parts of the text.
